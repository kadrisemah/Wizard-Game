üîß Troubleshooting

Install option not showing?

  • Check HTTPS: The site must be served over HTTPS (not HTTP)
  • Wait for service worker: Wait 5-10 seconds after page load
  • Try different browser: Use Chrome on Android or Safari on iOS
  • Clear cache: Clear browser cache and reload the page

App not working offline?

  • Load once online: You must load the app at least once with internet
  • Wait for caching: Service worker needs time to cache resources
  • Check console: Open developer tools to see service worker status

How to update the app?

  • Automatic updates: The app checks for updates automatically
  • Update prompt: You'll see "New version available! Reload to update?"
  • Manual update: Close and reopen the app to check for updates

How to uninstall?

  • Android: Long-press the app icon ‚Üí "Uninstall" or "Remove"
  • iOS: Long-press the app icon ‚Üí "Remove App" ‚Üí "Delete App"
  • Desktop: Right-click app icon ‚Üí "Uninstall" or go to system settings
